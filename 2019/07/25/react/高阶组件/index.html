什么是高阶组件

一个高阶组件只是一个包装了另外一个 React 组件的 React 组件。

高阶组件有什么用

  • 代码复用，逻辑抽象，抽离底层准备（bootstrap）代码
  • 渲染劫持
  • State 抽象和更改
  • Props 更改
